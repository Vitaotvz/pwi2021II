  • Github e Git Bash

    • No Github nós guardamos nossos projetos feitos com diversos tipos de linguagens de desenvolvimento, utilizamos o terminal Gitbash para enviar tais projetos.

    O Gitbash não é o único terminal disponível, existem vários outros! No terminal do Linux por exemplo a funcionalidade do Git é nativa.

    Linguagens Utilizadas

  • HTML

    • HTML é ó código utilizado para fazer um website, Basicamente, HTML é a linguagem utilizada para estruturar todo o conteúdo de um site.

    HTML não é uma linguagem de programação mas sim uma linguagem de marcação, pois ela apenas encapsula dados e descreve o que fazer com eles, não como fazer. Ela não é Turing Complete, ou seja, ela precisaria ter algumas características específicas para poder "programar um dispositivo"

  • CSS

    • CSS é o código utilizado para deixar o seu website bonito, é um código de estilização.

    É com a linguagem CSS que damos cor para o Título, configuramos o tamanho do texto, largura das colunas e etc.
